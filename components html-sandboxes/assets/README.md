# Design System Assets

Design tokens and reference materials for the AutomationAI3to5 Design System.

## Contents

- **tokens.json** - Design system tokens (colors, typography, spacing, shadows, etc.)
- **icons/** - 470+ SVG icons organized by category (see `../../react-ui/src/icons/`)
- **fonts/** - Typography reference

## Design Tokens

All token values are defined in `tokens.json` for easy integration with:
- CSS variables
- SCSS/LESS
- JavaScript frameworks (React, Vue, Angular)
- Design tools (Figma, Sketch, Adobe XD)

## Typography

Font Family: System font stack (-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto...)
Sizes: xs, sm, base, lg, xl, 2xl, 3xl, 4xl, 5xl
Weights: 400 (regular), 500 (medium), 600 (semibold), 700 (bold)

## Spacing Scale

4px baseline: xs (4px), sm (8px), md (16px), lg (24px), xl (32px), 2xl (48px), 3xl (64px)
