Installation & Setup

Installation

The Design System is available as an npm package:

npm install @automationai/design-system

Basic Usage

Link the CSS in your HTML:

<link rel="stylesheet" href="css/main.css">

Usage Examples

<button class="btn btn-primary">Click me</button>
<div class="card">Card content</div>

React Integration

import { Button, Card } from '@automationai/design-system/react';
